An NFC project · privacy-by-design

Every tap
leads back to you.

A portable kiosk that writes your social profile to an NFC keychain in seconds — so the people you meet at events can follow you with a single tap of their phone. No app. No cloud. No data trail.

01 What it is

A small, supervised station for trading follows in the physical world.

// privacy

Nothing is logged.

Your URL is held in memory just long enough to write it to a tag, then discarded. No accounts, no analytics, no trail.

// offline

No internet required.

The kiosk runs entirely on a Raspberry Pi at the table. It writes tags whether the venue Wi-Fi works or not.

// universal

Tap-to-follow, like tap-to-pay.

The recipient taps the keychain to their phone — no app install, no QR scan. The link opens. They follow.

02 Status

In active development.

The project began as an Arduino prototype with a character LCD and has since grown into a Raspberry Pi 4 system with a screen-based interface, USB keyboard input, and a USB PN532 NFC reader.

Phase 2 — the web UI running in Chromium kiosk mode — is currently being wired up. Hardware decisions and accessibility refinement are in flight.
